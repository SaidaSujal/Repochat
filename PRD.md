# RepoChat PRD

## Project
RepoChat — Chat with any public GitHub repository.

## Goal
Users paste a public GitHub repository URL and ask questions about the codebase.
Answers must be grounded in retrieved code and include citations.

## MVP Features
- Repository URL input
- Repository ingestion
- Code chunking
- Embeddings
- Vector database
- RAG chat
- File and line citations
- Error handling

## Out of Scope
- Private repositories
- Multi-agent systems
- Fine tuning
- Code execution
- Multi-repository chat
